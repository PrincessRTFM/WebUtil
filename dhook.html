Welcome to DHook!

This is a graphical (browser-based) interface to discord webhooks. It's built on discord.js as an API, and is currently in a pre-release state. To-be-implemented features include support for file uploads, custom label/content fields in the rich embed, and the ability to use uploaded files in place of any of the URL fields.

Hover over something on the interface to see some simple help text about it and how to use it on the right sidebar.

Quick Basics

You need to provide a webhook ID (you can paste the URL that discord gives you into the box, DHook can extract the ID) and some kind of content to post.

Rich embeds are the things that bots usually post, and are also basically what website preview embeds are. You can post an embed with any content you want, even none at all!
