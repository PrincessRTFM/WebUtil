Welcome to NikoMaker!

This tool allows you to quickly create an image of Niko (the main character of Oneshot) saying whatever you like, with any expression from the game!

Hover over an element to get more information on it.

Quick Basics

You need to select Niko's expression and write a message to display on the image.

Once you've done that, you can copy or save the created image by right clicking on it.
